### git init

> 初始化仓库,生成.git目录

### git add xx / git add .

> 将修改添加到暂存区

### git commit -m'change message'

> 将暂存区的内容纳入到版本控制中,并标注修改的简介

### git branch your-branch-name

> 创建分支

### git checkout branch-name

> 切换到指定的分支

### git checkout -b branch-name

> 创建并切换到指定的分支

### git branch -d branche-name

> 删除指定的分支

### git merge xxx

> 与 xx 进行合并

### git rebase xx

> 与 xx 进行合并，能得到线性的提交记录
